What's the Buzz about Bee Loss?

Decline of Bees

"U.S. National Agricultural Statistics show a honey bee decline from about 6 million hives in 1947 to 2.4 million hives in 2008, a 60 percent reduction."

Why are Bees declining?

The main reasons for global bee-decline are linked to industrial agriculture, parasites/pathogens and climate change. The loss of biodiversity due to monocultures and the wide-spread use of bee-killing pesticides are particular threats for honeybees and wild pollinators.

Tableau and Repository

Michael created the Tableau dashboard. It features several stories. These maps have a legend and are interactive. Several maps have an icon that we found to symbolize honey, bees or almond trees. Tableau is a powerful data visualization tool used in the Business Intelligence Industry.
Michael also created our timeline in google docs explaining our process and outlined ownership of each project component. He created the Readme and organized our repository.

HTML/CSS

Morgan built this website and it's animations using HTML and advanced CSS. Our website follows the BEM model with SASS properties. Morgan did her editing using a live NPM server for quick fixes. She also utilized the browser developer tools. It's scalable and has media queries to support phones to large desktops.
We have imbedded our Tableau dashboard and have interative buttons, clickable links, and a form to email each of the contributors to this project.
